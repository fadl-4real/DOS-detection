# 🛡️ DOS Detection System – Kali Linux

🚨 Built as part of my cybersecurity internship @ Uneeq Interns

This project implements a real-time Denial of Service (DOS) detection system using Python and Scapy on a Kali Linux VM. The system sniffs live traffic and triggers alerts when a SYN flood pattern is detected.

---

## 💡 Features

- ✅ Real-time packet sniffing using Scapy  
- ✅ SYN flood attack detection logic  
- ✅ Terminal-based live alerting  
- ✅ PCAP logging & Wireshark analysis  
- ✅ Compatible with low-resource VMs (tested on 16 GB RAM / i7 CPU)
